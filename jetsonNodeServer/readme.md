# Jetson Server

## Pre-requisites 

- NodeJS (4.0.0 and above)
- Python (2.7 and above)

## Installation

```bash 
$ git clone 
$ cd jestson-dl/jetsonNodeServer
$ sudo npm install
```

## Quick Start

Start the server:

``` bash
$ node jetsonServer.js

```
### Note:

Complete installation steps in **jestson-inference** for server to start processing images.
